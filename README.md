# TestCropPredictions
Short code to test the quality of crop production prediction models

To run, copy the repository and run the test_models.py. To indicate the location of the ground truth file, parse --ground-truth \<path_to_the\> file when running the code.

The predictions are already in the repository. The script is written under the assumption that the dataset file has the same structure as the pub_yipeeo_yield_fl, but with the yield values not null.
